# 🌟 انرژی هوشمند صنعتی — Opportunity Hub

## 📊 درباره پروژه

داشبورد حرفه‌ای برای **نظارت بلادرنگ** و **تحلیل مصرف انرژی** کارخانه‌های صنعتی.

### ✨ ویژگی‌های اصلی:

- ⚡ **مصرف انرژی بلادرنگ** - ۵ KPI پویا
- 🏭 **۱۰ دستگاه صنعتی** - شبیه‌سازی دینامیکی
- 📊 **نمودارهای تعاملی** - Chart.js
- 🚨 **سیستم هشدار هوشمند** - ۳ سطح شدت
- 🌍 **دوزبانه** - فارسی و انگلیسی
- 📱 **واکنشگرا** - موبایل، تبلت، دسکتاپ
- 🎨 **UI مدرن** - Glassmorphism + Dark Mode
- ⚙️ **بدون Backend** - Pure HTML/CSS/JavaScript

---

## 🎯 KPI‌های اصلی:

1. **کل مصرف** - مصرف لحظه‌ای به کیلووات
2. **انرژی انباشته** - کل تولید انرژی روزانه
3. **سهم تجدیدپذیر** - درصد انرژی سبز
4. **بهره‌وری** - تعادل مصرف و تولید
5. **هشدارها** - تعداد ناهنجاری‌ها
